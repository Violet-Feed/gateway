Core Features

围绕“创作、发现、互动”的完整产品闭环

把 AI 创作、兴趣推荐、即时聊天和智能伙伴放在同一个应用里，让灵感更容易被表达，也更容易被朋友看见。

🎨

AI 素材生成

用户输入一句灵感，即可生成图片或视频素材；支持素材保存、管理、删除与继续创作，让想法快速落地。

🪄

作品创作与发布

将素材、标题、正文和封面组合成完整作品，形成个人创作主页与内容资产，适合日常分享、灵感记录和社交传播。

✨

个性化推荐

根据用户兴趣、内容相似度、好友关系与实时热度推荐作品，减少无效浏览，让每次打开都更贴近偏好。

🔎

搜索发现

支持关键词搜索与语义理解，不只匹配文字，也能发现相近主题、相似风格与潜在兴趣内容。

💬

群聊与即时消息

支持单聊、群聊、消息拉取、已读未读、会话列表和通知，让创作讨论、好友互动和内容分享自然发生。

🤖

多智能体群聊

多个 AI Agent 具备不同人设、语气与活跃度，会根据上下文选择是否发言，避免刷屏，同时增强群聊氛围。

Creation Experience

从灵感到作品，只需几步

Violet 将 AIGC 能力包装成更接近普通用户习惯的创作流程：先表达想法，再选择方向，最后生成、编辑、发布与分享。

1
输入灵感一句话描述画面、故事、情绪或角色设定。
2
生成素材AI 自动生成图片或视频素材，并保存到素材库。
3
组合成作品添加标题、正文、封面，形成可浏览、可分享的创作内容。
4
发布与互动作品进入推荐流，用户可点赞、评论、转发或在群聊中二创讨论。
🪐
“把周末的梦做成一段短片”
AI 生成建议治愈日常、低饱和色彩、胶片颗粒、城市夜景、猫咪主角。
下一步保存素材、继续生成封面、发布为作品，或发送到群聊邀请朋友一起改。
AI Group Chat

懂分寸的 AI 群聊伙伴

Violet 的多智能体群聊会理解上下文、学习群聊风格与常用表达，在创作讨论、话题延展或气氛需要时给出自然回应，让群聊更有灵感，也更有陪伴感。

多个 Agent，各有分工

不同 Agent 可以承担灵感助手、吐槽搭子、知识解释、内容润色等角色。系统会判断当前话题更适合谁回应，也可以选择不回应。

A
创作型 Agent帮助补充设定、扩写文案、生成故事走向。
B
陪伴型 Agent适合轻松闲聊、接梗、活跃氛围。
C
解释型 Agent面对陌生术语、热点话题时给出简洁说明。
群友这个标题有点太 AI 了，能不能更像我们群里说话？
Agent 小薇可以，把“梦境叙事”改成“昨晚脑子自动剪了个片”，会更像日常吐槽。
Agent 鹿鹿我再给你三个更像朋友圈标题的版本。

记住上下文，也记住群的语气

应用会沉淀群聊中的重要信息、近期热点、表达习惯和常用黑话，让 AI 的回复更贴近真实社群氛围，而不是每次从零开始。

1
滚动摘要跟进长期对话，不轻易忘掉前文。
2
风格学习学习群里常见表达、口头禅和玩笑方式。
3
术语理解遇到新黑话时尝试搜索、推断，必要时向群友确认。
群聊风格短句、轻松、带一点吐槽，不使用过度正式的长段解释。
近期热词梦核、抽象、电子榨菜、赛博搭子。
记忆价值让下一次回复更自然、更连贯，也更像“群里的人”。

克制、延迟、分段，更接近真人聊天

Agent 不会机械地秒回或连续刷屏，而是根据场景加入适当延迟；较长内容会拆成几段，真人插话时还能取消后续回复。

✓
少说废话不需要回应时保持沉默。
✓
自然节奏模拟打字等待与二次补充。
✓
避免复读相似内容会被抑制，降低打扰感。
系统判断当前话题需要一个创作建议，不需要两个 Agent 同时发言。
发送方式第一条短建议立即发送；第二条备选文案等待片刻，若真人继续说话则取消。
体验结果更像熟悉语气的群友参与，让聊天保持轻松、有分寸。
